# 直播弹幕助手

为 Bilibili 直播间添加弹幕鼠标悬浮 +1 发送、收藏功能。

## 功能

### 侧边栏弹幕悬停
鼠标悬停在聊天历史面板的弹幕上时，显示操作浮窗，支持快速 +1 发送和收藏弹幕。

### 播放器弹幕悬停
鼠标悬停在播放器飞行的弹幕上时，弹幕动画冻结并显示操作浮窗。

> 注意：仅支持 HTML DOM 弹幕渲染模式，Canvas 渲染模式下无法使用。

### 弹幕收藏
通过侧边栏小组件管理收藏的弹幕，支持搜索和快速发送。

## 使用方法

1. 在设置中启用"侧边栏弹幕悬浮"或"播放器弹幕悬停"
2. 鼠标悬停在弹幕上，点击 +1 按钮快速发送
3. 点击收藏按钮将弹幕添加到收藏库
4. 通过侧边栏小组件查看和管理收藏
