# 仅搜索首页 · 使用说明

> 组件内部名称：`searchHome`  
> 显示名称：**仅搜索首页**  
> 分支：`feature/search-home`  
> PR：https://github.com/the1812/Bilibili-Evolved/pull/5659

在 B 站首页只保留**顶栏**和**页面中央的搜索框**，隐藏推荐流、频道栏等干扰内容，适合「打开首页只为搜东西」的使用习惯。

---

## 效果一览

启用后，访问 `https://www.bilibili.com/` 时：

| 保留 | 隐藏 / 处理 |
|------|-------------|
| 顶栏（消息、头像等入口） | 首页推荐流、分区内容 |
| 顶栏原版搜索（若仍显示） | 频道栏 |
| 页面中央放大的 LaunchBar 搜索框 | 动画横幅 / Logo / 静态 banner 图 |
| 可自定义的纯黑或图片背景 | 底部多余空白滚动区 |

中间搜索框使用脚本自带的 **LaunchBar**，可与「搜索推荐」「BV 号跳转」等搜索相关插件配合。

---

## 安装方式（推荐：从本 fork 直接装）

无需本地跑 dev-server。在已安装 [Bilibili Evolved](https://github.com/the1812/Bilibili-Evolved) 的前提下：

### 1. 复制安装链接

**jsDelivr（推荐，国内通常更稳）：**

```text
https://cdn.jsdelivr.net/gh/mintonight/Bilibili-Evolved@feature/search-home/registry/dist/components/style/home-redesign/search.js
```

**GitHub Raw：**

```text
https://raw.githubusercontent.com/mintonight/Bilibili-Evolved/feature/search-home/registry/dist/components/style/home-redesign/search.js
```

若 CDN 缓存未刷新，可用具体 commit 固定版本，例如：

```text
https://cdn.jsdelivr.net/gh/mintonight/Bilibili-Evolved@e2eb42d9a/registry/dist/components/style/home-redesign/search.js
```

（将 `e2eb42d9a` 换成该分支上最新包含 `search.js` 的 commit。）

### 2. 在 Evolved 中安装

1. 打开任意 B 站页面  
2. 点击 **Bilibili Evolved** 图标 → **设置**  
3. 找到 **安装组件 / 从链接安装**（不同版本文案可能略有差异）  
4. 粘贴上面的链接并确认安装  
5. 在组件列表中启用 **仅搜索首页**

### 3. 互斥组件请先关闭

请勿与下列组件同时开启：

- **极简首页**（`minimalHome`）  
- **清爽首页**（`freshHome`）

### 4. 打开首页验证

访问：

```text
https://www.bilibili.com/
```

应看到接近「顶栏 + 居中搜索」的极简效果。

## 选项说明

### 背景颜色

- 默认：`#000000`（纯黑）  
- 在组件选项中用颜色选择器修改  

### 背景图片（额外选项）

在组件的 **额外选项** 区域：

1. **背景图片 URL**  
   - 粘贴 `http(s)://...` 图片地址  
   - **点击输入框外失焦后生效**（不是边输边生效）  
2. **选择本地图片**  
   - 从本机选择图片  
   - 以 Base64 保存在设置中，单张不超过 **1024 KiB**  
3. **清除**  
   - 去掉背景图，只保留背景颜色  
4. 下方可预览当前图片  

图片会以 `cover` 方式铺在背景颜色之上。

## 日常使用提示

- **搜索**：用页面中央的 LaunchBar 输入关键词回车，或配合搜索类插件使用快捷跳转。  
- **更新组件**：若本 fork 更新了 `search.js`，可在 Evolved 组件更新/重装中再次使用同一链接；jsDelivr 若仍旧，可换 commit 版链接。  
- **关闭组件**：关闭「仅搜索首页」后，原版首页会恢复（以脚本卸载逻辑为准，建议刷新页面确认）。  
- **自定义顶栏**：可与「自定义顶栏」同时使用；中间搜索仍由本组件提供。

## 本地开发（可选）

仅在需要改代码、调试时：

```powershell
# 项目根目录
pnpm install
pnpm install --dir registry

# 启动开发服务
pnpm tsx dev-tools/dev-server/index.ts

# 另开终端：编译本组件
pnpm tsx dev-tools/dev-server/command.ts build component style/home-redesign/search
```

本地调试链接：

```text
http://localhost:23333/registry/dist/components/style/home-redesign/search.js
```

生产构建并写到磁盘（便于 push 到 fork 供在线安装）：

```powershell
pnpm exec tsx -e "import webpack from 'webpack'; import { buildByEntry } from './registry/webpack/config.ts'; const config = buildByEntry({ src: './registry/lib/components/', type: 'component', entry: './registry/lib/components/style/home-redesign/search/index.ts', mode: 'production' }); webpack(config, (err, stats) => { if (err) { console.error(err); process.exit(1); } console.log(stats.toString({ colors: true })); process.exit(stats.hasErrors() ? 1 : 0); });"
```

然后将 `registry/dist/components/style/home-redesign/search.js` 提交并推送到 `feature/search-home` 分支。

## 已知注意点

1. 仅在 B 站**主站首页**生效（`www.bilibili.com/`），视频页、动态页等不受影响。  
2. 会禁用首页「悬浮视频」预览，点击卡片会直接进视频页（与其它首页改造组件行为一致）。  
3. 顶栏上的部分横幅装饰会被隐藏，但消息、头像等入口应保持可用；若 B 站改版导致顶栏异常，可暂时关闭本组件。  
4. 上游合并 PR 前，请始终从 **本 fork 的 `feature/search-home` 分支** 安装上述 JS 链接。
